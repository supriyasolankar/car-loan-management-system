
# Car Loan Management System

## Description
Car Loan Management System is a full stack web application developed using Java, Spring Boot, React.js, and MySQL.
It helps manage customer loan applications and loan approval process.

## Features
- Customer loan application
- Admin approval and rejection
- REST API based backend
- CRUD operations using Spring Data JPA

## Technologies Used
- Java
- Spring Boot
- Spring Data JPA
- REST APIs
- React.js
- MySQL

## Tools
- Git
- GitHub
- Postman
- Eclipse IDE
